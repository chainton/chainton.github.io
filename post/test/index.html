Hugo搭建GitHub个人博客

May 17, 2019
1 min read

Windows环境

步骤

  • 安装Hugo
  • Hugo环境配置
  • 生成站点
  • 安装主题
  • 运行Hugo服务器
  • 发表测试文章
  • 生成静态网站
  • 推送至GitHub

> 安装配置和测试

前往Hugo Releases下载对应版本的Hugo安装文件（Hugo Releases：https://github.com/gohugoio/hugo/releases） 解压后得到的exe文件路径添加至系统PATH中，命令行查看Hugo版本以验证是否配置成功。命令如下：

hugo version
Hugo Static Site Generator v0.55.5-A83256B9 windows/amd64 BuildDate: 2019-05-02T13:04:07Z

> 生成站点

新建一个与Hugo配置目录一致的目录作为站点文件夹如sites，然后执行如下站点生成命令：

D:\hugo\blog>hugo new site blog

> 安装主题

注：此处使用下载主题包并解压至blog\theme目录下的方法，也可使用git命令直接clone Hugo官方主题：https://themes.gohugo.io/

> Run Hugo Server

出现以上 http://localhost:1313/ 地址则说明服务器成功跑起来了，在浏览器输入该地址即可。

> 发表第一个测试文章

hugo new post/first.md

命令行提示文件创建成功即可，然后使用编辑器编辑该markdown文件，记得将draft值修改为false。

> 静态网站生成

hugo --theme=hyde --baseUrl="https://chainton.github.io/"

将’chainton‘值改为你的GitHub名执行，结果是静态页面都会生成到 public 目录下。

> 推送至GitHub

切换至public目录后操作

D:\hugo\blog>cd public
D:\hugo\blog\public>git init
D:\hugo\blog\public>git add .
D:\hugo\blog\public>git commit -m "first"
D:\hugo\blog\public>git remote add origin https://github.com/chainton/chainton.github.io.git
D:\hugo\blog\public>git push -u origin master
